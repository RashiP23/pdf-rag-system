# PDF Question Answering System using RAG

A Retrieval-Augmented Generation (RAG) application that allows users to upload PDF files and ask questions based on document content.

## Features
- PDF Upload
- Text Extraction
- Text Chunking
- Embeddings
- Vector Database (FAISS)
- Similarity Search
- AI-generated Answers
- Streamlit UI

## Technologies Used
- Python
- Streamlit
- LangChain
- FAISS
- Hugging Face Transformers
- PyPDF2

## Run Project

```bash
streamlit run app.py
